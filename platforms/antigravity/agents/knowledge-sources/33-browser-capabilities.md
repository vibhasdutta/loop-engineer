# Browser Research Capabilities

These are the web research capabilities available to an AI agent when browsing:

- Read HTML
- Render JavaScript
- Execute JavaScript
- Read Network Requests
- Read API Calls
- Inspect DOM
- Parse Metadata
- Read JSON-LD
- Extract Structured Data
- Read OpenGraph Tags
- Read Twitter Cards
- Capture Screenshots
- OCR Images
- Download PDFs
- Extract Tables
- Infinite Scroll Support
- Login Session Support
- Cookie Management
