# Knowledge Sources Directory

This is a reference for the researcher agent. Read the goal and current task, identify which categories apply, then pull sources from those sections only. Ignore irrelevant categories — don't load noise.

---

## How to use this

1. Read the goal and task
2. Identify 2–5 relevant categories below
3. Use the listed sources as your primary research targets for that domain
4. For any category not listed here, use general web search to find the best source
