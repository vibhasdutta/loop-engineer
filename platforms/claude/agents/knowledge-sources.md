# Knowledge Sources Index

The `knowledge-sources/` folder contains 33 category files. You do not need to read all of them. Identify which categories are relevant to the goal and task, then read only those files.

## How to use

1. Read the goal and current task from PLAN.md
2. Match it to the goal types below
3. Read the listed category files from `knowledge-sources/`
4. Use those sources to guide your research
